# MindMate

MindMate is a full-stack mental health support chatbot designed to provide empathetic conversation, mood tracking, journaling, and crisis resources. It offers a soothing, user-friendly interface with smooth animations and a calming blue/green palette. MindMate integrates advanced AI to provide compassionate support while prioritizing user safety through crisis detection and emergency resource information.

---

## Features

- **Chatbot** with empathetic mental health support powered by OpenAI GPT-4.
- **Crisis detection** that immediately alerts users and provides hotline info.
- Predefined conversational flows such as daily check-in, mood rating, anxiety relief, sleep support, relationship stress, deep breathing exercises, positive affirmations, and journaling prompts.
- **Mood Journal**: Save daily mood ratings and journal entries.
- **Mood History** chart to visualize emotional trends over time.
- Responsive and accessible UI with calming colors and smooth animations.
- Sidebar navigation for easy access to Mood Tracker, Resources, and Settings.
- Disclaimer clarifying this is not medical advice.
- Anonymous mode supported with locally generated user IDs.
- Backend API with Node.js, Express, and MongoDB for data persistence.
- Frontend built with React, styled-components, react-spring, and recharts.

---

## Technology Stack

- **Backend:** Node.js, Express, MongoDB, Mongoose, Axios, dotenv
- **Frontend:** React, React Router, styled-components, react-spring, recharts, axios
- **Testing:** Jest, Supertest (backend), React Testing Library (frontend)

---

## Prerequisites

- **Node.js** (version 18 or higher recommended)
- **npm** (comes with Node.js)
- **MongoDB** instance (local or cloud e.g., MongoDB Atlas)
- OpenAI API key (or Softgen AI key) for chatbot responses
